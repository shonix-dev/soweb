Pythonのお部屋に戻る

リスト・辞書・タプル・集合

複数の値をまとめて扱うための型です。用途ごとに向いている型が違うので、使い分けが大切です。

4つの型の特徴

Pythonには、複数のデータをまとめて扱うための便利な型がいくつかあります。場面に応じて使い分けることで、コードが分かりやすくなります。

  • リスト: 順番あり、変更可能
  • 辞書: キーと値の組み合わせ
  • タプル: 順番あり、変更しにくい固定データ向き
  • 集合: 重複なし
numbers = [10, 20, 30]
student = {"name": "Soshi", "grade": 2}
point = (5, 8)
colors = {"red", "blue", "red"}

print(numbers[0])
print(student["name"])
print(point)
print(colors)

よく使う操作

numbers.append(40)
student["club"] = "tennis"

for n in numbers:
    print(n)

使い分けの目安

  • 順番どおり並べたいならリスト
  • 名前付きで管理したいなら辞書
  • 変更したくない組ならタプル
  • 重複を除きたいなら集合

それぞれの使い方の例

同じ「複数データ」でも、目的によって向いている型が異なります。

shopping_list = ["milk", "bread", "eggs"]
profile = {"name": "Soshi", "age": 20}
location = (35.0, 139.0)
tags = {"python", "web", "python"}

print(shopping_list)
print(profile["age"])
print(location)
print(tags)

学習のポイント

  • 迷ったら、まずはリストと辞書をしっかり使えるようにすると実用的です。
  • タプルは「変更しない組」、集合は「重複しない集まり」と覚えると整理しやすいです。
  • 中身を `for` で回して扱う場面が多いので、ループとセットで慣れるのがおすすめです。
