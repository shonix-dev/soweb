関数

関数は、同じ処理をひとまとまりにして名前を付けたものです。再利用しやすくなり、コードも読みやすくなります。

関数の定義

関数を使うと、同じ処理を何度も書かずに済みます。処理に名前を付けることで、プログラムの意味も読み取りやすくなります。 

def greet(name):
    print(f"こんにちは、{name}さん")

greet("Soshi")
  • `def` で関数を定義します。
  • かっこの中に引数を書きます。
  • 必要なら `return` で値を返せます。

戻り値

def add(a, b):
    return a + b

result = add(3, 5)
print(result)
`print` は表示するだけですが、`return` は関数の結果を外へ返します。

関数を使うメリット

  • 同じ処理をまとめられる
  • あとから修正しやすい
  • プログラムを部品ごとに考えられる
def calculate_price(price, tax_rate):
    return int(price * (1 + tax_rate))

print(calculate_price(1000, 0.1))

引数の役割

引数を使うと、同じ処理でも渡す値によって結果を変えられます。関数をより柔軟に再利用できるようになります。 

def introduce(name, hobby):
    print(f"{name}さんの趣味は{hobby}です")

introduce("Soshi", "読書")
introduce("Aki", "サッカー")

学習のポイント

  • 1つの関数に役割を詰め込みすぎない
  • 名前から何をする関数か想像できるようにする
  • 同じコードを2回以上書きそうなら関数化を考える
