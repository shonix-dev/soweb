条件分岐

条件分岐は、条件によって処理を変えるための仕組みです。Pythonでは `if`、`elif`、`else` を使って書きます。

if文の形

条件分岐は「もしこの条件ならこの処理をする」という書き方です。点数、年齢、入力内容などによって結果を変えたいときによく使います。 

score = 78

if score >= 80:
    print("合格ラインです")
elif score >= 60:
    print("もう少しで高得点です")
else:
    print("復習して再挑戦しましょう")
Pythonでは波かっこの代わりにインデントで処理のまとまりを表します。

比較演算子

条件式では、「比べる」ための記号や、複数条件をまとめるキーワードを使います。

  • `==`: 等しい
  • `!=`: 等しくない
  • `>`, `<`, `>=`, `<=`: 大小比較
  • `and`, `or`, `not`: 条件の組み合わせ
age = 19
has_id = True

if age >= 18 and has_id:
    print("入場できます")

条件分岐を使う場面

  • ログイン成功か失敗かを分ける
  • テストの点数で評価を変える
  • 入力された値が正しいか確認する
command = "save"

if command == "save":
    print("保存します")
else:
    print("別の処理を行います")

入れ子の条件分岐

条件分岐の中に、さらに別の条件分岐を書くこともできます。ただし増えすぎると読みにくくなるため、 まずは順番に条件を並べられないか考えるのが大切です。 

age = 20
has_ticket = True

if age >= 18:
    if has_ticket:
        print("入場できます")
    else:
        print("チケットが必要です")
else:
    print("年齢条件を満たしていません")

学習のポイント

  • 条件は「何を比べているか」がすぐ分かるように書く
  • 分岐が多いときは、上から順にどの条件が先に判定されるか意識する
  • 迷ったら `print()` で条件の値を確認しながら進めると理解しやすいです
