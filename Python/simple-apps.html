簡単なアプリ制作

学んだ文法を定着させるには、小さくても完成形のあるアプリを作るのが効果的です。入力、処理、出力を一通り経験できます。

まず作りやすいもの

  • じゃんけんゲーム
  • ToDo管理
  • 簡単な家計簿
  • クイズアプリ

ミニアプリ例

tasks = []

while True:
    task = input("追加するタスクを入力してください。終了は end: ")
    if task == "end":
        break
    tasks.append(task)

print("登録したタスク")
for item in tasks:
    print("-", item)

アプリ制作で身に付くこと

  • 入力、分岐、ループ、関数を組み合わせる力
  • データをどう持つか設計する力
  • 使いやすさやエラー処理を考える力

作るときの流れ

  • 何を入力するか決める
  • どんな処理をするか整理する
  • 結果をどう表示・保存するか考える

最初から大きいものを目指すより、1画面・1機能の小さなアプリを積み重ねる方が上達しやすいです。

まず動く最小版を作り、そのあとで保存機能や削除機能などを少しずつ追加していく進め方がおすすめです。
