SQL と Python

Pythonからデータベースを扱うと、保存・検索・更新の処理を自動化できます。学習では標準ライブラリの `sqlite3` が始めやすいです。

sqlite3 の基本

import sqlite3

conn = sqlite3.connect("school.db")
cursor = conn.cursor()

cursor.execute("CREATE TABLE IF NOT EXISTS students (name TEXT, grade INTEGER)")
cursor.execute("INSERT INTO students VALUES (?, ?)", ("Soshi", 2))
conn.commit()

データ取得

cursor.execute("SELECT name, grade FROM students")
rows = cursor.fetchall()

for row in rows:
    print(row)
SQL文を文字列結合で作るより、プレースホルダ `?` を使う方が安全です。

よくある操作

  • `INSERT`: データを追加する
  • `SELECT`: データを取得する
  • `UPDATE`: データを更新する
  • `DELETE`: データを削除する
cursor.execute(
    "UPDATE students SET grade = ? WHERE name = ?",
    (3, "Soshi"),
)
conn.commit()

Pythonは処理全体を制御し、SQLはデータの検索や更新条件を表す役割だと考えると整理しやすいです。
