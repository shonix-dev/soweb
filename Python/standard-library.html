Pythonのお部屋に戻る

標準ライブラリ活用

Pythonには最初から便利なライブラリがたくさん入っています。外部ライブラリを増やさなくても、日付処理やファイル操作などをかなりこなせます。

よく使う標準ライブラリ

  • `datetime`: 日付と時刻
  • `pathlib`: パスやファイル操作
  • `random`: 乱数
  • `math`: 数学関数
from datetime import datetime
from pathlib import Path
import random

now = datetime.now()
print(now.strftime("%Y-%m-%d"))

path = Path("sample.txt")
print(path.exists())

print(random.randint(1, 6))

ほかにも便利なもの

  • `collections`: 数え上げや便利なデータ構造
  • `itertools`: 繰り返し処理の補助
  • `os`: 環境変数やディレクトリ操作
  • `sys`: 実行引数やPython環境の取得
from collections import Counter

words = ["python", "web", "python", "data"]
count = Counter(words)
print(count)

活用の考え方

「まず標準ライブラリでできないか」を考えると、依存関係を増やしすぎずに済みます。学習初期にも扱いやすいです。

外部ライブラリを入れる前に、標準ライブラリに似た機能がないか調べる習慣を持つと、環境構築もシンプルになります。
