JSON / CSV の扱い

JSONはデータ交換でよく使われ、CSVは表形式データで広く使われます。Pythonではどちらも標準ライブラリで扱えます。

JSON

import json

data = {"name": "Soshi", "score": 90}

with open("data.json", "w", encoding="utf-8") as file:
    json.dump(data, file, ensure_ascii=False, indent=2)

`dump` はファイル保存、`load` は読み込みに使います。

with open("data.json", "r", encoding="utf-8") as file:
    loaded = json.load(file)

print(loaded["name"])

CSV

import csv

rows = [
    ["name", "score"],
    ["Soshi", 90],
    ["Aki", 85],
]

with open("scores.csv", "w", newline="", encoding="utf-8") as file:
    writer = csv.writer(file)
    writer.writerows(rows)
with open("scores.csv", "r", encoding="utf-8") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

実務での使いどころ

  • APIのレスポンスはJSONが多い
  • ExcelとやりとりするときはCSVが多い
  • データ保存形式ごとの特徴を知っておくと便利です

階層のあるデータや設定ファイルにはJSON、表形式の一覧データにはCSVが向いています。 どちらも実務でよく出るので、保存だけでなく読み込みにも慣れておくと役立ちます。
