Webスクレイピング

Webスクレイピングは、Webページから情報を取り出す技術です。調査や一覧取得に便利ですが、サイトの利用規約や負荷への配慮が必要です。

基本例

import requests
from bs4 import BeautifulSoup

url = "https://example.com"
html = requests.get(url, timeout=10).text
soup = BeautifulSoup(html, "html.parser")

title = soup.find("h1")
print(title.text)

複数要素を取る例

一覧ページでは、同じ種類の要素をまとめて取り出すことがよくあります。

items = soup.find_all("li")

for item in items:
    print(item.text.strip())

注意点

  • `robots.txt` や利用規約を確認する
  • 短時間に大量アクセスしない
  • ページ構造が変わるとコードも修正が必要です

APIとの違い

APIがあるならAPIを優先し、ない場合にページから情報を取得するのが一般的です。スクレイピングはHTML構造への依存が大きい点に注意します。

HTML構造が変わると取得結果が崩れることもあるので、定期実行する場合は失敗時に気付けるようログや確認処理を入れると安心です。
