文字列操作

文字列は、名前やメッセージなどを扱うための型です。つなげたり、一部分を取り出したり、書式を整えたりできます。

基本操作

文字列は `"` または `'` で囲んで書きます。つなげる、取り出す、長さを数えるといった操作が基本になります。 

first_name = "Soshi"
last_name = "Oniki"

full_name = first_name + " " + last_name
print(full_name)
print(full_name[0])
print(len(full_name))

便利なメソッド

text = "  python basic  "

print(text.strip())
print(text.upper())
print(text.replace("basic", "lesson"))

`strip()` は前後の空白削除、`upper()` は大文字化、`replace()` は文字の置き換えです。

f文字列

変数を自然な形で文の中に埋め込めます。

name = "Soshi"
score = 92

print(f"{name}さんの点数は{score}点です")

よくある活用例

  • ユーザー名やメッセージの表示
  • 入力データの整形
  • CSVやテキストの簡単な加工

部分文字列の取り出し

文字列は、必要な部分だけを取り出すこともできます。これをスライスと呼びます。 

text = "Python"

print(text[0])    # P
print(text[0:3])  # Pyt
print(text[-1])   # n

学習のポイント

  • 文字列も順番のあるデータとして扱えます。
  • `+` で連結、`len()` で長さ確認、メソッドで加工、という流れを押さえると便利です。
  • 入力値は文字列で受け取ることが多いので、実用場面でも頻繁に登場します。
