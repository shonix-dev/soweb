ループ

同じ処理を何度も繰り返したいときにループを使います。Pythonでは `for` と `while` が基本です。

for文

回数が決まっている繰り返しや、リストの中身を順に取り出すときに便利です。

for i in range(5):
    print(i)

fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

`range(5)` は 0 から 4 までの5回分を作ります。

while文

条件が成り立つ間だけ繰り返します。

count = 0

while count < 3:
    print(count)
    count += 1
  • `break`: 途中でループを抜ける
  • `continue`: その回だけ残りを飛ばして次へ進む
for number in range(1, 6):
    if number == 3:
        continue
    if number == 5:
        break
    print(number)

ループの活用例

ループは一覧表示、集計、繰り返し入力の処理などでよく使います。

scores = [80, 75, 92, 64]
total = 0

for score in scores:
    total += score

print(total)

ループと条件分岐の組み合わせ

ループの中で条件分岐を使うと、繰り返しながら必要なデータだけを選ぶことができます。 集計やフィルタの処理で特によく使います。 

numbers = [3, 8, 12, 15, 20]

for number in numbers:
    if number % 2 == 0:
        print(number)

学習のポイント

  • `for` は回数や対象が決まっているときに使いやすいです。
  • `while` は条件で止めたいときに向いています。
  • `while` では終了条件を書き忘れると無限ループになるので注意します。
