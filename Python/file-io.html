Pythonのお部屋に戻る

ファイル入出力

ファイル入出力では、テキストファイルを読み込んだり、結果を保存したりできます。Pythonでは `with open(...)` を使う書き方が定番です。

読み込み

読み込みでは、ファイルの内容をプログラムの中に取り込みます。テキストの確認、データの読み込み、ログの解析などで使います。 

with open("sample.txt", "r", encoding="utf-8") as file:
    content = file.read()

print(content)

書き込み

with open("result.txt", "w", encoding="utf-8") as file:
    file.write("Pythonの学習記録です\n")
    file.write("ファイルに文章を書けました")

`w` は新しく書き直すモードなので、既存の内容を消して保存します。

モードの違い

  • `r`: 読み込み
  • `w`: 上書き保存
  • `a`: 追記
`with` を使うと、処理が終わったあと自動でファイルが閉じられるので安全です。 
with open("log.txt", "a", encoding="utf-8") as file:
    file.write("1行追加しました\n")

行ごとに読み込む

長いファイルでは、一度に全部読むより1行ずつ扱う方が分かりやすいことがあります。 

with open("sample.txt", "r", encoding="utf-8") as file:
    for line in file:
        print(line.strip())

学習のポイント

  • まずは `with open(...)` の形を覚えると実用的です。
  • 文字化けを防ぐために `encoding="utf-8"` を意識すると安心です。
  • 上書き保存の `w` は既存内容を消すので、最初は特に注意します。
