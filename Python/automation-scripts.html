自動化スクリプト

毎回同じ手順で行う作業は、Pythonで自動化できることがあります。ファイル整理、データ変換、定期処理などと相性がよい分野です。

自動化の例

  • フォルダ内のファイル名を一括変更する
  • CSVをまとめて整形する
  • 毎日同じレポートを生成する
from pathlib import Path

folder = Path("reports")

for path in folder.glob("*.txt"):
    new_name = path.stem + "_backup.txt"
    path.rename(folder / new_name)

自動化しやすい作業の見つけ方

  • 毎日・毎週繰り返す作業
  • 手順は単純なのに件数が多い作業
  • 人のミスが起こりやすい定型作業
from pathlib import Path

for csv_file in Path("data").glob("*.csv"):
    print(csv_file.name)

自動化するときの考え方

最初は小さな作業から始めるのが安全です。ファイル操作は取り返しがつかないこともあるため、テスト用データで確認してから本番に使います。

  • 処理前後のログを残す
  • 削除や上書きは特に慎重に扱う
  • 最初はコピーを使って試す
