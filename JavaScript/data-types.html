JavaScriptの部屋に戻る

データ型

データ型は、その値がどんな性質を持つかを表します。JavaScriptでは数値、文字列、真偽値などをよく使います。

主な型

  • `number`: 数値
  • `string`: 文字列
  • `boolean`: `true` と `false`
  • `null`: 意図的に空
  • `undefined`: 値が未定義
const age = 20;
const name = "Soshi";
const isStudent = true;
let memo;
const result = null;

console.log(typeof age);
console.log(typeof name);
console.log(typeof isStudent);

文字列と数値の違い

見た目が同じでも、型が違うと結果も変わります。

console.log(10 + 5);    // 15
console.log("10" + 5);  // 105
フォーム入力などでは文字列として受け取ることが多いので、型を意識することが大切です。

型変換

必要に応じて、文字列を数値に変換したり、数値を文字列にしたりします。

const textNumber = "25";
const number = Number(textNumber);

console.log(number + 5);
console.log(String(number));

学習のポイント

  • まずは `number`、`string`、`boolean` をしっかり押さえる
  • `typeof` を使うと型の確認ができます
  • 入力値を扱うときは、文字列のままか数値へ変換するかを意識します
