JavaScriptの部屋に戻る

関数

関数は、ひとまとまりの処理に名前を付けたものです。同じ処理を何度も書かずに済むので、コードを整理しやすくなります。

基本形

function greet(name) {
  console.log(`こんにちは、${name}さん`);
}

greet("Soshi");

戻り値

function add(a, b) {
  return a + b;
}

const result = add(3, 5);
console.log(result);

引数の役割

引数を使うと、同じ処理でも渡す値によって結果を変えられます。再利用しやすい関数を書くうえで大切です。

function introduce(name, hobby) {
  console.log(`${name}さんの趣味は${hobby}です`);
}

introduce("Soshi", "読書");
introduce("Aki", "サッカー");

学習のポイント

  • 同じコードを繰り返し書きそうなら関数化を考える
  • 1つの関数に役割を詰め込みすぎない
  • 名前から何をする関数か分かるようにする
表示するだけの関数と、値を返す関数の違いを意識すると理解しやすいです。
