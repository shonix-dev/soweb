オブジェクト

オブジェクトは、名前付きのデータをひとまとめにするための仕組みです。人や商品など、属性を持つものを表すのに向いています。

基本

オブジェクトは、1つのものに関する情報をまとめるための入れ物です。 たとえば「生徒」なら、名前、学年、部活のように複数の情報を1つにできます。 

const student = {
  name: "Soshi",
  grade: 2,
  club: "tennis",
};

console.log(student.name);
console.log(student["grade"]);

`student.name` は「student の中の name を取り出す」という意味です。

プロパティの追加

student.score = 90;
console.log(student);

配列が順番で管理するのに対して、オブジェクトは名前で管理します。

  • 配列は「何番目か」で取り出すことが多いです
  • オブジェクトは「name」や「price」のような名前で取り出します
  • データの意味が分かりやすいので、実際のWeb開発でもよく使われます

使う場面

人、商品、記事のように、名前付きの属性をまとめて持たせたいときにオブジェクトが向いています。

const product = {
  name: "ペン",
  price: 120,
  stock: 10,
};

console.log(`${product.name}は${product.price}円です`);

学習のポイント

  • 順番重視なら配列、名前付き管理ならオブジェクト
  • `.name` と `["name"]` の両方の書き方があります
  • 配列の中にオブジェクトを入れる形もよく使います
