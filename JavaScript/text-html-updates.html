テキストやHTMLの書き換え

要素を取得したあとは、その中身を書き換えられます。表示文字を変えたり、新しい HTML を差し込んだりすることで、ページの見た目を動的に更新できます。

textContent

`textContent` は、要素の文字だけを安全に書き換えたいときに使います。

const message = document.querySelector("#message");
message.textContent = "保存が完了しました";

innerHTML

`innerHTML` は HTML を含めて書き換えられます。便利ですが、扱いには注意も必要です。

const result = document.querySelector("#result");
result.innerHTML = "<strong>重要</strong>なお知らせ";
外部から受け取った文字列をそのまま `innerHTML` に入れるのは危険なことがあります。まずは `textContent` を優先するのが安全です。

使う場面

検索結果を表示する、送信完了メッセージを出す、ボタンのラベルを変えるなど、画面内容を変えたい場面で使います。 

const button = document.querySelector("#send-button");
button.textContent = "送信中...";

学習のポイント

  • 文字だけ変えるなら `textContent`
  • HTMLごと差し替えるなら `innerHTML`
  • 更新前にどの要素を変えたいのかを明確にする
