条件分岐

条件分岐は、条件によって実行する処理を変える仕組みです。JavaScriptでは `if`、`else if`、`else` を使います。

if文

条件分岐は「もしこの条件ならこの処理をする」という考え方です。点数、入力内容、ボタン操作などによって動作を変えたい場面でよく使います。 

const score = 78;

if (score >= 80) {
  console.log("高得点です");
} else if (score >= 60) {
  console.log("合格です");
} else {
  console.log("復習しましょう");
}

使う場面

  • 入力値のチェック
  • ボタン操作による分岐
  • ログイン成功・失敗の表示
const command = "save";

if (command === "save") {
  console.log("保存します");
} else {
  console.log("別の処理を行います");
}

入れ子の条件分岐

条件の中にさらに条件を書くこともできますが、増えすぎると読みにくくなるので注意します。

const age = 20;
const hasTicket = true;

if (age >= 18) {
  if (hasTicket) {
    console.log("入場できます");
  } else {
    console.log("チケットが必要です");
  }
} else {
  console.log("年齢条件を満たしていません");
}

学習のポイント

  • どの条件が先に判定されるかを意識する
  • 長くなりすぎる条件は変数に分けると読みやすい
  • 迷ったら `console.log()` で条件の中身を確認する
