配列

配列は、複数の値を順番にまとめて管理するための仕組みです。一覧表示や繰り返し処理でよく使います。

基本操作

const fruits = ["apple", "banana", "orange"];

console.log(fruits[0]);
fruits.push("grape");
console.log(fruits.length);

繰り返し

const scores = [80, 75, 92];

for (const score of scores) {
  console.log(score);
}

よく使う操作

const tasks = ["study", "exercise"];

tasks.push("sleep");
tasks.pop();

console.log(tasks.includes("study"));
console.log(tasks);

学習のポイント

  • 配列は順番のあるデータに向いています
  • 添字は 0 から始まります
  • ループと組み合わせて使う場面がとても多いです
