演算子

演算子は、計算や比較、条件の組み合わせを行うための記号です。数値計算だけでなく、条件分岐でもよく使います。

算術演算子

const a = 10;
const b = 3;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
console.log(a % b);

比較と論理演算子

  • `===`, `!==`: 厳密な比較
  • `>`, `<`, `>=`, `<=`: 大小比較
  • `&&`, `||`, `!`: 条件の組み合わせ
const age = 19;
const hasId = true;

console.log(age >= 18 && hasId);
console.log(age === 19);
console.log(!hasId);

JavaScriptでは `==` より `===` を使う方が安全です。

代入演算子

変数に値を入れたり、今の値をもとに更新したりするときに使います。

let count = 0;

count += 1;
count += 2;

console.log(count);

学習のポイント

  • 計算用と比較用の演算子を区別する
  • 条件分岐では `===` を優先する
  • 複数条件は `&&` と `||` を使って組み合わせる
