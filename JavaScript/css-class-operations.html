JavaScriptの部屋に戻る

CSSクラスの操作

JavaScript から CSS クラスを付けたり外したりすると、見た目を動的に変えられます。表示・非表示や色の切り替えでよく使います。

classList

const box = document.querySelector(".box");

box.classList.add("active");
box.classList.remove("hidden");
box.classList.toggle("open");

よくある使い方

メニューの開閉やエラー表示などで、クラスの付け替えがよく使われます。

const menu = document.querySelector("#menu");

function openMenu() {
  menu.classList.add("is-open");
}

状態の切り替え

`toggle()` を使うと、同じボタンで表示と非表示を切り替えるような処理を書きやすくなります。

const panel = document.querySelector("#panel");
const button = document.querySelector("#toggle-button");

button.addEventListener("click", function () {
  panel.classList.toggle("is-hidden");
});

学習のポイント

  • 見た目の変更は CSS、切り替えのきっかけは JavaScript という役割分担が分かりやすいです
  • `toggle()` はオン・オフの切り替えに便利です
  • スタイルを直接変更するより、クラス操作の方が管理しやすいです
