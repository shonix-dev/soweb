フォーム操作

フォーム操作では、入力欄や送信ボタンから値を受け取り、JavaScript で確認や加工を行います。ユーザー入力を扱う基本としてとても重要です。

入力値の取得

フォームでは、入力欄に入った値を受け取って画面表示やチェックに使います。

const input = document.querySelector("#user-name");
const button = document.querySelector("#check-button");

button.addEventListener("click", function () {
  console.log(input.value);
});

送信時の処理

フォーム送信では、`submit` イベントと `preventDefault()` をよく使います。

const form = document.querySelector("#profile-form");

form.addEventListener("submit", function (event) {
  event.preventDefault();
  console.log("送信前のチェックを行います");
});
`preventDefault()` を使うと、送信直後の画面遷移を止めて、JavaScript で入力チェックしやすくなります。

入力チェックの例

空欄かどうかを調べるだけでも、フォーム操作の基本練習になります。

const input = document.querySelector("#user-name");
const error = document.querySelector("#error-message");

if (input.value.trim() === "") {
  error.textContent = "名前を入力してください";
}

学習のポイント

  • 入力欄の値は `value` で取得します
  • 送信前チェックはフォーム操作の基本です
  • 空欄確認や文字数確認など、小さなバリデーションから始めると理解しやすいです

フォーム操作は、要素取得、イベント処理、テキスト更新がまとまって登場するので、DOM操作の総合練習としてもとてもよい題材です。
