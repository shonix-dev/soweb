変数と定数

JavaScriptでは、データを一時的に保存するために変数や定数を使います。値に名前を付けることで、処理の意味が分かりやすくなります。

基本の書き方

`let` はあとで値を変えられる変数、`const` は再代入しない値に使います。

let score = 80;
const userName = "Soshi";

score = 90;

console.log(score);
console.log(userName);

使い分け

  • 値が変わるなら `let`
  • 基本は `const` を優先すると意図が分かりやすい
  • `var` は古い書き方なので、学習初期は `let` と `const` を中心に覚えると十分です
const taxRate = 0.1;
let price = 1200;

price = price * (1 + taxRate);
console.log(price);

使う場面

変数や定数は、入力された値、計算途中の結果、設定値などを保存するときに使います。 値に名前を付けることで、コードを読んだときに意味が伝わりやすくなります。 

const itemName = "ノート";
let quantity = 3;
const pricePerItem = 120;

const total = quantity * pricePerItem;
console.log(`${itemName}の合計金額は${total}円です`);

学習のポイント

  • 名前は役割が分かるものにする
  • 再代入しないなら `const` を使う習慣を付ける
  • 1文字だけの名前ばかりにしない
`const` は「中身が絶対に変わらない」ではなく、「その変数名に別の値を再代入しない」という意味です。
