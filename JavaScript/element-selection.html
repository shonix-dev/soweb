要素の取得

DOM操作の最初の一歩は、HTML要素を取得することです。取得した要素に対して、文字の変更やイベント追加を行います。

querySelector

`querySelector()` は、CSSセレクタの書き方で要素を1つ取得できます。最初に覚える方法としてとても便利です。 

const title = document.querySelector("h1");
const button = document.querySelector(".save-button");
const input = document.querySelector("#user-name");

複数要素の取得

`querySelectorAll()` を使うと、条件に合う複数要素をまとめて取得できます。

const items = document.querySelectorAll("li");

for (const item of items) {
  console.log(item.textContent);
}

よくある取得パターン

  • `#id名`: 特定の1つを選ぶ
  • `.class名`: 同じ種類の要素を選ぶ
  • `タグ名`: `p` や `button` など要素名で選ぶ
const title = document.querySelector("#page-title");
const cards = document.querySelectorAll(".card");
const button = document.querySelector("button");

学習のポイント

  • まずは `querySelector()` と `querySelectorAll()` を使えるようにすると実用的です
  • 取得したい要素に `id` や `class` を付けると選びやすくなります
  • 要素が見つからないと `null` になるので、必要に応じて確認します
const button = document.querySelector("#save-button");

if (button) {
  console.log("ボタンが見つかりました");
}
