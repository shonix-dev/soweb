JavaScriptの部屋に戻る

DOMとは

DOM は、HTML を JavaScript から扱いやすくするための仕組みです。JavaScript は DOM を通して、見出しやボタンや入力欄などを取得し、内容を変えたり反応を付けたりできます。

DOMの考え方

ブラウザは HTML をそのまま読むのではなく、要素の親子関係を持ったデータとして扱います。 その構造を JavaScript から操作できるようにしたものが DOM です。 

<body>
  <h1>タイトル</h1>
  <p>説明文</p>
</body>

このような HTML は、JavaScript から見ると「body の中に h1 と p がある構造」として扱われます。

最初は DOM を「画面そのもの」ではなく、「画面を作るための設計図を JavaScript で触れる形にしたもの」と考えると整理しやすいです。

DOMでできること

  • 要素を取得する
  • 文字を変える
  • CSSクラスを付け替える
  • クリック時の動きを追加する
const title = document.querySelector("h1");
title.textContent = "新しいタイトル";

`document.querySelector("h1")` で要素を見つけて、`textContent` で中の文字を変えています。

DOMを使う場面

DOM は、ボタンを押したときに文字を変える、メニューを開閉する、入力結果を画面に表示するなど、 Webページに動きを付ける場面で広く使われます。 

const message = document.querySelector("#message");

message.textContent = "読み込みが完了しました";

学習のポイント

  • `document` はページ全体を表す入口です
  • まずは「要素を取得して文字を書き換える」流れを覚えると理解しやすいです
  • DOM操作は Webページに動きを付ける基礎になります
JavaScriptだけで画面を変えているように見えても、実際には DOM の中の要素を変更していると考えると整理しやすいです。
