文字列操作

文字列は、名前やメッセージなどを扱うための型です。つなげたり、一部分を取り出したり、形を整えたりできます。

基本操作

const firstName = "Soshi";
const lastName = "Oniki";

const fullName = firstName + " " + lastName;
console.log(fullName);
console.log(fullName.length);

テンプレートリテラル

const name = "Soshi";
const score = 92;

console.log(`${name}さんの点数は${score}点です`);

便利なメソッド

const text = "  javascript basic  ";

console.log(text.trim());
console.log(text.toUpperCase());
console.log(text.replace("basic", "lesson"));

部分文字列の取り出し

文字列の一部だけを取り出したいときは `slice()` などを使います。

const text = "JavaScript";

console.log(text[0]);
console.log(text.slice(0, 4));
console.log(text.slice(-6));

学習のポイント

  • 文字列も順番のあるデータとして扱えます
  • 連結、長さ確認、置換の3つは特によく使います
  • 画面表示やフォーム入力で頻繁に登場します
