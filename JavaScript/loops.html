JavaScriptの部屋に戻る

ループ

ループは、同じ処理を繰り返すための仕組みです。JavaScriptでは `for` と `while` が基本です。

for文

for (let i = 0; i < 5; i += 1) {
  console.log(i);
}

回数が決まっている繰り返しに向いています。

配列との組み合わせ

const fruits = ["apple", "banana", "orange"];

for (const fruit of fruits) {
  console.log(fruit);
}

while文

let count = 0;

while (count < 3) {
  console.log(count);
  count += 1;
}
終了条件を更新しないと無限ループになるので注意します。

ループと条件分岐の組み合わせ

繰り返しながら条件で絞り込む処理は、一覧表示や集計でよく使います。

const numbers = [3, 8, 12, 15, 20];

for (const number of numbers) {
  if (number % 2 === 0) {
    console.log(number);
  }
}

学習のポイント

  • `for` は回数が決まっているときに使いやすい
  • `for...of` は配列の中身を順番に扱うときに便利です
  • `while` は終了条件の更新忘れに注意します
