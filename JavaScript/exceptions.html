例外処理

例外処理は、エラーが起きたときに安全に対応するための仕組みです。JavaScriptでは `try`、`catch`、`finally` を使います。

基本

try {
  const jsonText = "{ name: Soshi }";
  const data = JSON.parse(jsonText);
  console.log(data);
} catch (error) {
  console.log("JSONの読み込みに失敗しました");
}

finally

try {
  console.log("処理を開始します");
} catch (error) {
  console.log("エラーが発生しました");
} finally {
  console.log("処理を終了します");
}
例外処理はエラーを隠すためではなく、利用者に分かりやすく安全に処理を続けるために使います。

よくある使いどころ

  • JSONの変換
  • 外部データの読み込み
  • 想定外の値に対する安全対策
try {
  throw new Error("手動でエラーを発生させました");
} catch (error) {
  console.log(error.message);
}

学習のポイント

  • 全部を `try` で囲むのではなく、失敗しそうな部分に絞る
  • `catch` では何が起きたか分かるメッセージを出す
  • 例外処理は「失敗しても安全に終わらせる」ために使います
