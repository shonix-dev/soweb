JavaScriptの部屋に戻る

属性の取得と変更

HTML要素には `src`、`href`、`alt`、`value` などの属性があります。JavaScript から属性を読み取ったり、書き換えたりできます。

getAttribute と setAttribute

const link = document.querySelector("a");

console.log(link.getAttribute("href"));
link.setAttribute("href", "https://example.com");

プロパティで扱う例

よく使う属性は、直接プロパティとして扱えることもあります。

const image = document.querySelector("img");

image.src = "photo.jpg";
image.alt = "風景の写真";

使う場面

画像の切り替え、リンク先の変更、入力欄への初期値設定などで属性操作が使われます。 

const link = document.querySelector("#profile-link");

link.href = "https://example.com/profile";
link.textContent = "プロフィールを見る";

学習のポイント

  • リンク先や画像の差し替えでよく使います
  • `getAttribute()` は属性値の確認に便利です
  • フォームでは `value` のようなプロパティも頻繁に使います
