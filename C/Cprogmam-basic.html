プログラミングの基礎

ここでは、コードプログラミングにおける基本的な事柄について学んでいきます。
この内容は、どのプログラミング言語にも共通することです。知っておいて損はありません。

プログラミングとは？

プログラミングとは、コンピュータに「何を、どのように実行するか」を命令として書き表すことです。
人間が「やりたいこと」をコンピュータが理解できる形（プログラム）に翻訳する作業、とも言えます。
コンピュータは、自分で考えることができません。与えられた命令通りにしか動きません。
そこで人間は、 コンピュータ専用の言語を使って、命令を送ることで、コンピュータを動かしています。

Main関数 / Mainメソッド

C言語（C, C++）では、関数定義を除き、基本的にすべての命令を main関数 の中に記述します。

C#では、基本的には Mainメソッド の中に命令を記述します。
Mainメソッド は、他の言語でいう main関数 と同じ役割を持ちます。

C言語の場合


#include <stdio.h>

int main() {
    //処理内容をここに書く。
    return 0;
}

C++の場合


#include <iostream>

int main() {
    //処理内容をここに書く。
    return 0;
}

C#の場合


using System; 

class Program
{
    static void Main(string[] args) 
    {
        //ここに処理を書く
    }
}
