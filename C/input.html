変数と標準入出力

目標

  • 変数の概要を理解し、正確に使用できる。
  • C言語における標準入出力のやり方を理解できる。

変数について

C言語（C/C++/C#）では、変数を宣言する際、変数の型を指定したうえで宣言する必要があります。
具体的にはこんな感じです。
下はCのプログラム例ですが、C++やC#でも同様に定義できます。

変数宣言


    #include <stdio.h>

    int main() {
        int a = 10; //数値型
        char b[100] = "おにぎり"; //文字型
        return 0;
    }

変数の型の種類

代表的な変数の型を表形式にまとめます。

型の名前 意味
int型 数値（主に整数）を扱う。
char型 単体では、1バイトの文字列を扱う。
[]内に数値をいれると複数の文字列を扱える。
floot型,
double型		 小数を扱う。
bool型 真偽値（true/false）を扱う
