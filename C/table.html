Learning Map

C言語 / C++ / C# の学習トピック

3つの言語を横並びで見比べながら、どこが共通していて、どこから発展していくのかをつかめるように整理しました。

C言語

基礎力を育てる

変数、入出力、条件分岐、繰り返しなど、プログラミングの土台をまっすぐ学ぶのに向いています。

C++

Cの理解を広げる

C言語の基礎を踏まえつつ、より大きなプログラム設計に進むための入口として整理できます。

C#

アプリ開発へつなげる

開発環境の準備から、クラスや継承まで含めて、実践的な学習の流れが見える構成です。

学習の見方

C言語とC++は基礎トピックがかなり共通しています。一方でC#は、環境構築やオブジェクト指向の学習項目が早い段階から入ってくるのが特徴です。
