標準入出力について

標準入出力とは、コンピュータが外部とやり取りをするための基本的な仕組みです。
主に、キーボードからの入力を受け取ったり、コンピュータが結果を出力する際に使われるプログラムのことを指すことが多いです。

変数

C言語（C/C++/C#）では、変数を宣言する際、変数の型を指定したうえで宣言する必要があります。
具体的にはこんな感じです。

変数宣言


using System;

class Program
{
    // C#のエントリポイント（プログラムの開始点）
    static void Main(string[] args)
    {
        // 宣言のみ
        int number; 

        // 値の代入 (初期化)
        number = 100; 
        
        // 代入された値の確認
        Console.WriteLine($"number の値: {number}");

        // 宣言と同時に初期化
        string message = "Hello, C#!";
        
        // 初期化された値の確認
        Console.WriteLine($"message の値: {message}");
        
        // その他の型の例
        double pi = 3.14159;
        bool isTrue = true;
        
        Console.WriteLine($"pi の値: {pi}");
        Console.WriteLine($"isTrue の値: {isTrue}");
    }
}

コーディングでのプログラミングにおいて、変数はよく、箱に例えられることが多いです。
箱の中に数値や文字、記号など、様々なものを出し入れするようなイメージをもつと良いと思います。
